Qualification sessions

Repeat the run. Measure the noise. Package the proof.

Import a `chromofold.qualification-session.v1` file containing at least three matched baseline and candidate runs. Analysis remains in your browser and can be saved locally or exported.

Local storage is used only after you explicitly save. Nothing is uploaded.

Import or load a repeated-run session to begin.

Decision rule

Repeatability is part of correctness.

At least three runs

Single-run comparisons remain useful for exploration but cannot establish repeatability.

Median comparison

Qualification uses medians so one warm-up or scheduling outlier does not decide the result.

Variability gate

Any metric above 5% coefficient of variation makes the session incomplete.

Regression visibility

A material latency or throughput regression remains visible even when memory improves.

ChromoFoldSession PASS is portable evidence, not maintainer qualification.
